      field guide 🌼 (what the words mean)
      

        
A Flower program is three parts: declarations, a garden block, and a logic block.


int width = 6;          // how wide the garden is
int depth = 6;          // how deep
enum garden[depth][width];
int bouquet = 0;

{                       // 1st block: build the garden
  (1, 3) putFlower ;    // (x, y) — column, row
  (0, 0) putPicker ;
  (5, 5) putExit ;
}

{                       // 2nd block: the stroll
  (2) moveRight ;       // walk 2 cells right
  ( ) pickFlower ;      // pick whatever is underfoot
}

        
    
          
  • moveUp / moveDown / moveLeft / moveRight (n) — walk n cells; rocks and the hedge stop you.
    • 
          
  • pickFlower ( ) — pick the flower you're standing on. bouquet goes up by one.
    • 
          
  • putFlower / putGrass / putBarrier / putPicker / putExit (x, y) — place things.
    • 
          
  • check ( ) — what's underfoot · peek ( ) — what's one cell ahead.
              empty 0 · flower 1 · grass 2 · rock 3 · gate 4
    • 
          
  • giveUp ( ) — sit down among the weeds.
    • 
          
  • for(i < 3) { ... } loops while the condition holds; if / else and break work as you'd hope.
    • 
          
  • After every move, x and y hold where the picker stands.
    • 
        

        
Tip: pick a stamp on the right and click the garden — the construction block writes itself.
